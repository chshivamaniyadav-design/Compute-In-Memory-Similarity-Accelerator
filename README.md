# Compute-In-Memory Accelerator for Similarity Search

## Overview

This project implements a Compute-In-Memory (CiM) architecture using Verilog HDL. The design performs bitwise logical operations directly inside SRAM memory arrays and uses a Popcount unit for similarity detection.

## Features

- 128-bit Parallel Processing
- SRAM-based Compute-In-Memory
- AND, OR, XOR and XNOR operations
- Popcount Similarity Detection
- FPGA Synthesizable
- Low Power Design

## Tools Used
